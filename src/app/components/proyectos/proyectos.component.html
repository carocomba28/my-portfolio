Proyecto de gestión de notas

Este proyecto cuenta con un login, un CRUD de notas y la posibilidad de filtrar por categoría y por notas archivadas o desarchivadas.

  • Tecnologías para el fronted: Angular, Bootstrap.
  • Tecnologías para el backend: Java, Spring Boot.
Proyectos varios con HTML, CSS, JavaScript.

Incluye un cronómetro, el juego Snake, una calculadora y un sistema de Cifrado César.

Proyecto de carrito de supermercado

En este proyecto, se puede elegir entre varios frutos secos y añadirlos al carrito. Mientras se van agregando productos, se puede ver cómo se suma el total de la compra. Y si se cambia de opinión, también se puede sacar productos del carrito.

Proyecto final del curso de Java

Este proyecto consiste en un sistema que realiza cálculos para determinar el sueldo neto de un empleado. Incluye funciones para calcular descuentos, asignaciones familiares y un plus por antigüedad, considerando el cargo y la cantidad de hijos del empleado.
