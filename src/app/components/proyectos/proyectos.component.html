Proyecto de gestión de notas

Este proyecto cuenta con un login, un CRUD de notas y la posibilidad de filtrar por categoría y por notas archivadas o desarchivadas. Además guarda la información en una base de datos MySQL.

  • Tecnologías para el frontend: Angular, Bootstrap.
  • Tecnologías para el backend: Java, Spring Boot.

Proyecto final del curso de Java

Este proyecto consiste en un sistema que realiza cálculos para determinar el sueldo neto de un empleado. Incluye funciones para calcular descuentos, asignaciones familiares y un plus por antigüedad, considerando el cargo y la cantidad de hijos del empleado.
